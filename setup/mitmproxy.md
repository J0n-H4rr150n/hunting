# mitmproxy  

## Install  

install pipx:  
```
sudo apt update && sudo apt install pipx -y
```

install on ubuntu with pipx:  
```
pipx install mitmproxy
```  

ensurepath:  
```
pipx ensurepath
```  

check if it works:  
```
mitmproxy
```
